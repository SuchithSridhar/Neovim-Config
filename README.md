# Neovim-Config

This repository contains my neovim configuration written in lua.

Some details:

- `Treesitter` used for syntax highlighting.
- `Telescope` used for fuzzy file finding and more.
- `Bufferline` used to keep track of different buffers.
- `NerdTree` used to navigate directories in nvim.
- `LuaLine` used for better status line at the bottom.

More details may be found in comments in the individual config files.
